# IMDb Sentiment Analysis using Machine Learning

## Project Overview
This project performs sentiment analysis on IMDb movie reviews using Natural Language Processing (NLP) and Machine Learning techniques.

The goal is to classify reviews as **Positive** or **Negative** based on review text.

---

## Dataset
- IMDb Movie Reviews Dataset
- Sentiment labels created using review ratings:
  - Rating ≥ 7 → Positive
  - Rating ≤ 4 → Negative

---

## Techniques Used
- Text cleaning & preprocessing
- TF-IDF vectorization
- Machine Learning models:
  - Logistic Regression
  - Naive Bayes
  - Support Vector Machine (SVM)
  - Random Forest
- Hyperparameter tuning (GridSearchCV)
- Class imbalance handling using SMOTE
- Model evaluation using Accuracy, F1-score, Confusion Matrix

---

## Best Results
- **SVM Accuracy:** ~93%
- Logistic Regression performed competitively with good interpretability.
